1절. 리스트 : lst = [타입과 관련없는 0개 이상의 데이터들]
	len(lst) : 요소 개수
	min(lst), max(lst), sum(lst) : 최소, 최대, 합
	data in lst : lst 안의 data 요소가 있는지 여부
	list[idx], lst[from:stop:step] 인덱싱과 슬라이싱
	추가 : lst.append(value), lst.insert(idx, value)
	삭제 : lst.pop(), lst.pop(idx), lst.remove(data)
	찾기 : lst.index(data)_없으면 에러남, lst.count(data)_data가 몇개 있는지. 없으면 0
